Instagram Graph API


Home Page


1. Setup your app for Instagram Graph API.
Open documentation of Instagram Graph API and follow the instructions to setup.
Click here to open Getting Started documentation.
The API should return a User access token.
Capture the below credentials so your app can use it in the next few queries.
a) Get client id (App id) from facebook app
b) Get client secret from facebook app
c) Get User Access Token from Graph API tool.

2. Get the credentials from step 1 and paste it in defines.py file and run the app again.

3. By default, Instagram User Access Tokens are short-lived and are valid for one hour. However, short-lived tokens can be exchanged for long-lived tokens.
Click here to get Long Live Access Token.
The API should return a Long Live User access token. Capture the token so your app can replace it with the access token fetched in step 1.

4. Get User's Pages
Click here to get User's Pages.
Capture the page id of user's page which is connected to Instagram Business Account and paste it in defines.py file.

5. Get the Page's Instagram Business Account
Click here to get Page's Instagram Business Account.
This should return the IG User — an Instagram Business or Creator Account — that's connected to the Facebook Page.
Capture the id of Instagram Business or Creator Account and paste it in defines.py file.

6. Get the Instagram Business Account's Media Objects
Click here to get the Instagram Business Account's Media.
This should return id, caption, media_type, media_url, permalink, thumbnail_url, timestamp, username, comments_count, like_count, children{media_url, media_type, thumbnail_url}, comments{username, timestamp, text, id}
